# Spotify History

A Streamlit app for exploring your personal Spotify listening history.
Upload your exported Spotify data and get insights into your top artists, albums, tracks, and listening patterns over time.

Inspired by the original Streamlit app by [Tyler Simons](https://share.streamlit.io/user/tyler-simons).

## Usage

Clone the repo and install the requirements:
`pip install -r requirements.txt`

Then you can run the app
`streamlit run spotify_history.py`
